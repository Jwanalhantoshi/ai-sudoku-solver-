# AI Sudoku Solver

An intelligent Sudoku-solving system that uses heuristic algorithms to solve puzzles quickly and efficiently through an interactive graphical interface.

---

# Features

- Heuristic-based Sudoku solving
- OCR puzzle detection from images
- Interactive GUI interface
- Step-by-step solving visualization
- Multiple difficulty levels
- Random puzzle generation
- Solve / Pause / Reset controls
- Speed control options

---

# Tools Used

- Python
- Tkinter
- CustomTkinter
- OpenCV (cv2)
- NumPy
- Pytesseract OCR

---

# Algorithm Used

- Heuristic Algorithm (MRV Heuristic)

The heuristic technique helps the system choose the best cell first by selecting the cell with the fewest possible values. This reduces unnecessary search steps and improves solving speed.

---

# GUI Features

- Upload Sudoku puzzle images
- Load predefined random puzzles
- Solve puzzles automatically
- Visualize the solving process
- Pause and reset solving
- Change solving speed

---

# Project Workflow

1. Load or upload a Sudoku puzzle
2. Read and analyze the puzzle
3. Detect empty cells
4. Apply heuristic solving rules
5. Solve the puzzle automatically
6. Display the final solved board

---

# Future Improvements

- Validate puzzle selection before solving
- Enable reset during solving
- Improve OCR for different image qualities
- Support more Sudoku puzzle types
